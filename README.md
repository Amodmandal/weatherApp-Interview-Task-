# Flutter Weather App

This is a Flutter Weather App built with:

- BLoC state management
- Dio API calls
- WeatherAPI.com integration
- GetStorage for local persistence
- GPS & location permissions handling
- Splash / Help screen
- Save/Update button with reactive behavior
- Test widget included

## Features
- Help screen with skip & auto redirect
- Weather by city or GPS
- Save location using GetStorage
- Clean BLoC architecture
